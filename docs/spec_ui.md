# UI Specification

## Telas

### Login
- Campo de email
- Campo de senha
- Botão de login

### Dashboard
- Lista de tarefas
- Campo para nova tarefa
- Botão de adicionar
- Checkbox para concluir tarefa

## Fluxo
Login → Dashboard → Gerenciamento de tarefas
