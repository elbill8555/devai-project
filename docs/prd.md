# Product Requirements Document (PRD)

## Objetivo
Desenvolver uma aplicação web de gerenciamento de tarefas com autenticação.

## Funcionalidades principais
- Cadastro e login de usuário
- Criar tarefas
- Listar tarefas
- Marcar como concluída
- Deletar tarefas

## Requisitos não funcionais
- Interface simples e intuitiva
- Responsividade
- Performance adequada

## Métricas de sucesso
- Usuário consegue criar e gerenciar tarefas facilmente
